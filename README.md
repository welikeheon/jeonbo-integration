## Setup
### Prerequisites ###

* PHP = 7.2
* MariaDB = 10.4

### Install

1. Install `composer.phar`
2. Execute `composer install`

### Configuration

1. Copy to `/config/config.php` file and fill in values, the format is as same as in `/config/config.example.php`
